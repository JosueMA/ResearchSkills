## Additional information

#### To Install LaTeX on your computer

* A latex distribution
  * Windows: MikTex
  * Mac: MacTex https://tug.org/mactex/
  * Linux: LaTex (often already installed)
* An editor
  * TexStudio (what we use in class)
  * TexMaker
  * ...
